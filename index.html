‚ö° What is LeUnBrIck?

LeUnBrIck.sh is a terminal-based bash script to recover bricked iPod Nano devices using DFU/WTF modes, Apple firmware, and tools like wInd3x and ipodscsi.

üì• How to Use

  1. Clone the repository: git clone https://github.com/lycanld/LeUnBrIck
  2. Change directory: cd LeUnBrIck
  3. Make it executable: chmod +x LeUnBrIck.sh
  4. Run the script: ./LeUnBrIck.sh
  5. Follow the terminal UI to select your model and flash firmware

üìÅ Features

  • Automatic detection of DFU/WTF iPods
  • Auto-install required tools (Option 3)
  • Firmware flashing support for:
    • iPod Nano 7G (2012)
    • iPod Nano 7G (2015)
    • iPod Nano 6G
  • Auto-fetch firmware files from AppleDB.dev if missing
  • Supports flashing via wInd3x and ipodscsi (Linux/macOS)
  • Color-coded terminal UI
