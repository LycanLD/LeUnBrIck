🎧 LeUnBrIck

Universal Unbricker & Flasher for iPod Nano 6G & 7G (2012 / 2015)

View on GitHub

⚡ What is LeUnBrIck?

LeUnBrIck is an all-in-one toolkit to restore bricked iPod Nano 6G & 7G devices. It supports both 2012 & 2015 models and works cross-platform with Linux, macOS, and Windows (BETA).

📥 How to Use

Linux / macOS

  1. Clone the repository: git clone https://github.com/lycanld/LeUnBrIck.git
  2. Change directory: cd LeUnBrIck
  3. Make it executable: chmod +x LeUnBrIck.sh
  4. Run the script: ./LeUnBrIck.sh
  5. Follow the terminal UI to select your model and flash firmware

Windows (BETA)

  1. Clone the repository: git clone https://github.com/lycanld/LeUnBrIck.git
  2. Double-click LeUnBrIck_Windows.bat or run in Command Prompt
  3. The script will check Python, install requests if missing, and run main.py
  4. Follow on-screen instructions to enter DFU/WTF mode and flash firmware

⚠️ Ensure your iPod is connected before running.

📁 Features

  • Restore iPod Nano 6G & 7G (2012 & 2015)
  • Flash WTF & firmware images safely
  • Auto-download missing .MSE firmware files
  • Color-coded terminal UI
  • Automatic installation of required tools on Linux/macOS
  • Windows Support (BETA)
